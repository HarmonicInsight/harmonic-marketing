# harmonic-marketing

HARMONIC insight のマーケティングコンテンツ管理リポジトリ。

## PPTX ナレーション品質ルール（必須）

PPTXのスピーカーノート（ナレーション/字幕テキスト）を作成・修正した場合は、以下のルールを厳守し、**必ず** `/check-narration` を実行してチェックすること。

### 改行・文字数ルール

| 項目 | 基準 |
|------|------|
| 1行の文字数 | 20〜45文字 |
| 文字カウント | 全角=1文字、半角=0.5文字 |
| 20文字未満の行 | 前後の行と結合する |
| 45文字超の行 | 自然な区切りで改行する |
| 1スライドの行数 | 7〜15行が目安 |

**自然な改行ポイント（優先順）:**
1. 句点「。」の後
2. 読点「、」の後
3. 閉じ括弧「）」「」」の後
4. 「「」「（」の前
5. 助詞「は」「が」「を」「に」「で」「と」の後

**例外:** 段落冒頭の短い導入文は20文字未満でもOK。英語技術用語を含む行は多少の超過を許容。

### 不要タグの除去

`【ナレーション】` `【台本】` `[Narration]` があれば除去する。

### XML属性（TTS用）

全ての `<a:rPr>` に以下を設定:
```
kumimoji="1" lang="ja-JP" altLang="en-US" dirty="0"
```
`lang="en-US"` のみはTTSエラーの原因→修正必須。

### 段落構造（XML）

- 意味的な段落ごとに `<a:p>` で分割
- 段落内の行は `<a:br>` で改行
- 行ごとに `<a:r>` + `<a:t>` で格納
- 1つの `<a:t>` に全文を入れない

## ブランド表記

- 正式名称: **HARMONIC insight**（HARMONICは大文字、insightは小文字）
- Harmonic Insight, HARMONIC Insight 等は誤り
