```

│  [受講者が社内で評価]              [研修実績が信頼に]          │

│                              [HARMONIC insight も提案]        │

└─────────────────────────────────────────────────────────────┘
```

**なぜ教材→BtoB研修が効くか:**
- **同じコンテンツで単価10〜30倍**: 個人¥4,980の教材 → 法人研修¥100,000〜300,000/回

- **導入支援との親和性**: 研修→「実際に導入しませんか？」が自然な流れ

### 5.4 STORES 活用戦略（個人向け教材・テンプレートの販売拠点）

- 手数料が安い（フリープラン: 5%、スタンダード: 3.6%）

- 日本語ネイティブのUI/UXで購入ハードルが低い
- クレジットカード・コンビニ決済・キャリア決済に対応

**STORES出品ラインナップ:**

## 6. スライド共有プラットフォーム戦略

### 6.1 位置づけ: 認知チャネルとしてのスライド共有

YouTube動画用に制作済みの **40本のPPTXスライド** は、動画の素材としてのみ活用されている。
これをスライド共有プラットフォームに公開することで、**追加の制作コストなし**でファネル上部（認知）を拡大できる。

```
┌─────────────────────────────────────────────────────────────┐
│          スライド共有の戦略的ポジション                         │
│                                                             │
│  [既存PPTX 40本]                                            │
│       │                                                     │
│       ├── YouTube動画（現状）                                │
│       │     └→ 視聴者は「動画を見る人」                      │
│       │                                                     │
│       ├── スライド共有（追加）★                              │
│       │     └→ Google検索・SNS経由の新規流入                 │
│       │     └→ note記事・YouTube動画への送客                 │
│       │     └→ 法人研修の信頼性向上（「教える側」の証明）     │
│       │                                                     │
│       └── note記事への埋め込み（相乗効果）                   │
│             └→ 記事のリッチ化 + 滞在時間UP                   │
│                                                             │
│  コスト: ほぼゼロ（PDF変換 + メタデータ設定のみ）             │
│  リスク: 低（社外秘チェックのみ必要）                         │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 プラットフォーム比較と選定

| プラットフォーム | 強み | 弱み | 推奨度 |
|----------------|------|------|--------|
| **SpeakerDeck** | シンプルUI、エンジニア・技術者に人気、埋め込み対応、無料 | アナリティクスが弱い | ★★★ |
| **Docswell** | 日本国内で成長中、日本語UIネイティブ、アナリティクス充実、リード獲得機能 | 海外リーチは弱い | ★★★ |
| **SlideShare** | SEO最強（ドメインパワー）、ビジネス層リーチ、LinkedIn連携 | LinkedIn未活用の場合効果半減、UIが重い | ★★☆ |

**推奨構成: SpeakerDeck + Docswell の2プラットフォーム並行運用**

- **SpeakerDeck** → 技術系コンテンツ（AI実務活用、LLM比較）+ 海外リーチ
- **Docswell** → ビジネス系コンテンツ（ビジネス基礎、コンサル論）+ 国内リード獲得

SlideShareはLinkedIn活用を開始するPhase 3以降に検討。

### 6.3 公開対象スライドの優先度と出し分け

**全40本を一度に出すのではなく、シリーズ×プラットフォームで戦略的に出し分ける。**

#### 優先度 高（Phase 2で即公開）

| シリーズ | 本数 | 公開先 | 理由 |
|---------|------|--------|------|
| ビジネス基礎 | 5本 | Docswell | 汎用性が高く検索流入を狙える。ビジネス層がターゲット |
| AI実務活用 | 5本 | SpeakerDeck + Docswell | 技術×ビジネスの両面。最も検索ボリュームが大きい |
| LLM比較 | 4本 | SpeakerDeck | 技術者コミュニティで話題性あり。鮮度が命なので早期公開 |

#### 優先度 中（Phase 2後半〜Phase 3）

| シリーズ | 本数 | 公開先 | 理由 |
|---------|------|--------|------|
| コンサル論 | 4本 | Docswell | ソートリーダーシップ確立。法人研修への導線 |
| AIショート | 6本 | SpeakerDeck | 短く鋭い主張で拡散力あり。Twitter連動向き |
| AIセキュリティ | 1本 | SpeakerDeck + Docswell | 法人ニーズ高、両プラットフォームで |

#### 優先度 低（慎重に判断）

| シリーズ | 本数 | 判断 | 理由 |
|---------|------|------|------|
| 製品紹介（ずんだもん等） | 3本 | 非公開推奨 | 宣伝色が強く、共有サイトでは逆効果 |
| ローカルシリーズ | 8本 | 選択公開 | ニッチだが刺さる層には深い。技術系2-3本に絞る |
| 人事・研修 | 2本 | Docswell限定 | 法人研修への直接導線として活用 |
| 動画化実務 | 1本 | 非公開推奨 | 自社ツール前提で汎用性が低い |

### 6.4 スライド公開時の共通ルール

1. **社外秘チェック**: 顧客固有データ・社内限定情報の除去を確認
2. **ブランディング統一**: 全スライドの表紙・末尾にHARMONICロゴ + 連絡先を挿入
3. **CTA挿入**: スライド末尾に以下を追加
   - 関連YouTube動画へのリンク
   - 関連note記事へのリンク
   - 法人向け問い合わせ先（insight案件のみ）
4. **メタデータ設定**: タイトル・説明文・タグを日本語+英語で設定（SEO対策）
5. **PDF化**: PPTX → PDF に変換してアップロード（レイアウト崩れ防止）

### 6.5 期待効果

| 指標 | 根拠 |
|------|------|
| Google検索経由の新規流入 | SpeakerDeck/Docswellはドメインパワーが強く、スライドタイトルで上位表示されやすい |
| note記事・YouTube動画への送客 | スライド概要欄のリンクから既存コンテンツへ流入 |
| 法人研修の信頼性向上 | 「40本のスライドを公開している専門家」は研修提案時の説得力になる |
| 教材販売（STORES）へのファネル | 無料スライドで品質を体験 → 有料教材への転換 |
| Twitter/noteとの相乗効果 | スライド公開をネタにSNS投稿 → 拡散 → スライド閲覧数UP |

### 6.6 SNS連動サイクル（スライド公開時）

## 8. コンサル紹介チャネルの強化

### 8.1 現状の問題

「紹介してくれるが、広がらない」の根本原因:

1. **コンサルのインセンティブ不足** — 紹介しても自分の利益にならない
2. **説明コスト** — コンサルが製品を正確に説明できない
3. **顧客の温度感** — コンサル経由だと「売り込まれた」感が出る

### 8.2 解決策: パートナープログラム

```
┌──────────────────────────────────────────────┐
│          HARMONIC パートナー制度               │
│                                              │
│  ◆ 紹介パートナー                             │
│    紹介 → 成約で初年度売上の 25% をフィー      │
│    紹介ツール: LP + 2分デモ動画 + FAQ         │
│                                              │
│  ◆ 導入パートナー                             │
│    導入支援を代行 → 粗利の 40% が報酬         │
│    認定制度あり（研修1日 + 認定試験）          │
│                                              │
│  ◆ OEMパートナー                              │
│    製品を自社ブランドで再販 → 卸値 60%        │
│    大手SI向け                                 │
│                                              │
└──────────────────────────────────────────────┘
```

### 8.3 コンサル向け営業キット

social-poster で自動生成できるようにする:

- **2分デモ動画スクリプト**（INMV で動画化も可能）
- **1枚提案書**（顧客に見せる用）
- **ROI計算テンプレート**（「年間〇〇日削減」が出る）
- **FAQ集**（よくある質問と回答）
