## エグゼクティブサマリー

HARMONIC insight は現在、**B2B直販 + コンサル紹介**を主チャネルとしているが、
スケーラビリティに課題がある。本戦略では以下の3本柱で年商5,000万円を目指す:

1. **個人向けツール販売**（新ブランド）で認知とキャッシュフローを確保
2. **SNSコンテンツマーケティング**（既存social-poster）でファネル上部を拡大
3. **ツール→法人転換**のボトムアップ導線で B2B 売上を加速

```
┌─────────────────────────────────────────────────────────────┐
│                    マーケティングファネル                       │
```

## 1. ブランド戦略: 二層構成

### 1.1 ブランド体系

```
HARMONIC（親ブランド）
  ├── HARMONIC insight（法人向け）
  │     Insight Business Suite
  │     対象: エンタープライズ / 中堅企業
  │     価格: BIZ ¥49,800〜 / ENT 個別見積
  │
  └── HARMONIC tools（個人・小規模事業者向け）★新設
        実用ツールコレクション
        対象: 個人 / フリーランス / 小規模事業者
        価格: ¥0〜¥9,800（買い切り or 年額）
```

### 1.2 なぜ「HARMONIC tools」か

- **親ブランドとの繋がり**: 検索時に相互発見される
- **法人転換時の信頼**: 「あのツールと同じ会社の法人版」という安心感
- **完全別ブランドのリスク回避**: ゼロからの認知構築が不要
- **SEO効果**: "HARMONIC" で検索すると両方出てくる

### 1.3 ブランドガイドライン

| 要素 | HARMONIC insight | HARMONIC tools |
|------|-----------------|----------------|
| トーン | プロフェッショナル・信頼 | カジュアル・親しみやすい |
| 色 | Gold + Ivory | Gold + 白 + アクセントブルー |
| タグライン | AI で業務を、もっと確かに。 | 仕事が片づく、小さな道具箱。 |
| SNSアカウント | @harmonic_insight | @harmonic_tools |
| noteアカウント | harmonic-insight | harmonic-tools |

## 6. コンサル紹介チャネルの強化

### 6.1 現状の問題

「紹介してくれるが、広がらない」の根本原因:

1. **コンサルのインセンティブ不足** — 紹介しても自分の利益にならない
2. **説明コスト** — コンサルが製品を正確に説明できない
3. **顧客の温度感** — コンサル経由だと「売り込まれた」感が出る

### 6.2 解決策: パートナープログラム

```
┌──────────────────────────────────────────────┐
│          HARMONIC パートナー制度               │
│                                              │
│  ◆ 紹介パートナー                             │
│    紹介 → 成約で初年度売上の 25% をフィー      │
│    紹介ツール: LP + 2分デモ動画 + FAQ         │
│                                              │
│  ◆ 導入パートナー                             │
│    導入支援を代行 → 粗利の 40% が報酬         │
│    認定制度あり（研修1日 + 認定試験）          │
│                                              │
│  ◆ OEMパートナー                              │
│    製品を自社ブランドで再販 → 卸値 60%        │
│    大手SI向け                                 │
│                                              │
└──────────────────────────────────────────────┘
```

### 6.3 コンサル向け営業キット

social-poster で自動生成できるようにする:

- **2分デモ動画スクリプト**（INMV で動画化も可能）
- **1枚提案書**（顧客に見せる用）
- **ROI計算テンプレート**（「年間〇〇日削減」が出る）
- **FAQ集**（よくある質問と回答）
