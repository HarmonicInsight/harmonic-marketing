|---|---|---|

| **ビジネス特許** | 特開2020-107153（ロボット管理システム） | 技術的独自性の証明 |

### 2.2 一言で言えるブランドメッセージ

> **「28年企画してきた人間が、AIで全部自分で作る時代になった」**

このメッセージが刺さる理由:
- **共感**: 40〜50代の企画職・管理職が「自分もそうかも」と思える
- **具体性**: Kindle 40冊、140リポ、STORESという数字がある
- **希望**: 年齢関係なく、AIで新しいキャリアが開けるという前向きなストーリー
- **権威性**: 電通総研・PwC・FPTという看板が裏付ける

## 4. プラットフォーム別の役割と戦略

```
┌─────────────────────────────────────────────────────────────┐
│                    コンテンツ配信フロー                        │
│                                                             │
│  [YouTube Shorts] ← 認知獲得・フック                         │
│    「50代で140リポ作った」「企画屋が最強」                      │
│       ↓ チャンネル登録                                       │
│  [YouTube 本編] ← 深い理解・信頼構築                          │
│    キャリアストーリー、実演デモ、SIPO解説                       │
│       ↓ 概要欄リンク                                         │
│  [note 無料記事] ← SEO流入・権威性                            │
│    方法論の公開、建設DX知見                                    │
│       ↓ 有料記事/マガジン誘導                                 │
│  [note 有料記事] ← 収益化                                    │
│    Kindle出版術、SIPO完全版、企画書テンプレ                     │
│       ↓ STORES/HARMONIC insight誘導                          │

│  [HARMONIC insight] ← B2B・コンサル                           │
│       ↓ 実績化                                               │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 7. まとめ: なぜこの戦略が機能するか

1. **資産が既にある**: 40冊・140リポ・STORES・28年のキャリアは「これから作る」ではなく「既にある」。コンテンツ化するだけで成立する
2. **数字が語る**: 700社・80社・4.5億・40冊・140リポ。抽象論ではなく具体的な数字がある
3. **ファネルが自然に繋がる**: Shorts→本編→note→有料記事→STORES/コンサルという導線が無理なく設計できる
4. **競合がいない**: 「28年の上流経験 × AI × 40冊 × 140リポ × 実際に売っている」というポジションは市場にほぼ存在しない
5. **既存のマーケティング基盤**: social-poster、media-factory、video-managerなどのツールが既にある
