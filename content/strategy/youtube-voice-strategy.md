# YouTube ボイス・キャラクター戦略

> 策定日: 2026-03-29
> ステータス: 承認済み

## 背景

現在YouTube動画を20本以上アップロード済み。チャンネルのさらなる成長と将来のキャラクターチャンネル展開を見据え、ボイス・キャラクター戦略を策定する。

---

## チャンネルの哲学・コアメッセージ

### 基本スタンス：生成AI反対派

創業者は**生成AIの野放し利用に反対**の立場をとる。これはAI製品を作ることと矛盾しない。

> **「業務は作業手順の集合である。再現できないAI活用は、業務改善ではない。」**

### なぜ反対か

- 業務とは**作業手順の集合**である
- 作業手順は**再現可能**でなければならない
- 生成AIをそのまま使うと**毎回違う出力**になる → 作業手順にならない
- 属人的なプロンプト技術に依存する「AI活用」は**組織にスケールしない**

### HARMONIC insightの回答

AI Officeは生成AIを**作業手順の部品**に変換する：
- プロンプトをテンプレート化・標準化
- SIPOフレームワーク（Source→Input→Process→Output）で再現性を担保
- 誰がやっても同じ品質の出力を保証
- 「AIを使っている」意識なく業務が改善される

### 競合との決定的な差別化

```
× 他社:「ChatGPTにこう聞けばこう返ってくる」→ 知識で止まる、再現性なし
○ 当社:「このボタンを押せば同じ結果が出る」 → 業務が変わる、再現性あり
```

### 小説40冊を書く創業者だからこそ

- 小説はAIでは書けない — 人間の創造性・物語る力の証明
- AI Officeは創造的な仕事に集中するための道具
- **「人間がやるべきことと、機械に任せるべきことを分ける」**がメッセージの本質

## 戦略方針：3層ハイブリッド構造

1つのボイス/キャラに統一するのではなく、**マーケティングファネルの段階に応じて使い分ける**。

```
[認知]  → ずんだもん（エンタメ性・検索流入）
[興味〜検討] → アルカナオリジナルキャラ（親しみ＋専門性）
[転換]  → 青山流星 or 本人の声（権威性・信頼）
```

---

## 第1層：ずんだもん — 集客・認知（ファネル上部）

### 役割
- YouTube検索での集客力を活用
- 新規チャンネル登録者の獲得

### 対象コンテンツ
- AIセキュリティ・AI活用Tips系ショート動画
- 製品紹介の導入編（VID-050〜052の既存路線を継続）

### 実績のある動画フォーマット（ショート）

初期ショート動画で確立済みの構成：

```
[前半：掛け合いパート] ずんだもん × 四国メタン
  → 会話形式で興味を引く・わかりやすく解説
[後半：製品説明パート] 青山流星
  → 信頼感のある声で具体的な機能・価値を紹介
```

エンタメで掴み、プロの声で落とす——ファネル構造が1本の動画内に凝縮されたフォーマット。
四国メタン（VOICEVOX）はずんだもんとの掛け合いで定番の組み合わせであり、
「ずんだもん 四国メタン」での検索流入も期待できる。

### 運用方針
- 月4本のショート動画を目標
- 上記の掛け合い＋製品説明フォーマットを標準テンプレートとして継続
- 「ずんだもん」「四国メタン」タグを活用した検索流入の最大化
- ずんだもん・四国メタンは「入口」であって「チャンネルの顔」ではない

---

## 第2層：アルカナオリジナルキャラ — チャンネルの顔・ブランドIP

### 役割
- チャンネルのメインナビゲーター
- 独自IPとしてキャラクターチャンネルへの展開基盤

### キャラクター設計

既存のアーキタイプ（4天使エージェント）をベースに展開：

| キャラクター | 役割 | YouTube上での担当 |
|-------------|------|------------------|
| **ガブリエル（メイン）** | メッセンジャーの天使 | メインナビゲーター・解説役 |
| ミカエル | 戦略の天使 | ビジネス戦略・市場分析コンテンツ |
| ラファエル | 癒しの天使 | 品質・レビュー系コンテンツ |
| ウリエル | 光の天使 | ツール紹介・開発裏話 |

### なぜガブリエルをメインに据えるか
- 「お知らせの天使」→ YouTube解説役として最適
- 女性キャラクターとして親しみやすい
- 他の3天使はゲスト出演で掛け合いコンテンツが可能

### 展開ロードマップ
1. ガブリエルのキャラデザ＋VRM化
2. テスト動画2本で反応を検証
3. アルカナチャンネルとして本格稼働（4天使の掛け合い動画）

### アドバンテージ
- **IP所有権**: ずんだもんと違い、グッズ展開・有料コンテンツ化が自由
- **VRM連携**: `avatar-proactive-reactions.ts` の17種リアクションを動画キャラに転用可能
- **製品連携**: 各天使が担当製品を紹介 → キャラ＝製品理解

## 第3層：青山流星 / 本人の声 — 権威性・B2B向け

### 役割
- B2B商談につながる信頼性の担保
- 高単価コンテンツ（コンサル論・セミナー）

### 対象コンテンツ
- コンサルティング論
- キャリア系（28年の経歴を活かす）
- セミナー告知・登壇
- note有料記事の告知、BIZライセンス訴求

## 顔出し方針

### 基本方針: 限定的に導入（急がない）

キャラクターチャンネルとの共存を考え、顔出しチャンネルにはしない。顔は「スパイス」として使い、チャンネルの主役はコンテンツとキャラクターとする。

### 顔出しが効果的な場面（将来的に導入）
- チャンネル紹介動画（1本）
- セミナー・ウェビナー登壇映像
- お客様事例・対談インタビュー
- B2B向け動画の冒頭15秒＋締め10秒

### 顔出し不要な場面（大多数）
- 製品デモ・チュートリアル → 画面キャプチャ＋キャラ音声
- AI Tips・セキュリティ解説 → ずんだもん/アルカナキャラ
- ショート動画 → キャラクターの方がCTR高い

### 推奨フォーマット（導入時）
```
冒頭15秒：顔出し挨拶（信頼）
本編：画面キャプチャ＋キャラ音声（制作効率◎）
締め10秒：顔出しCTA（行動喚起）
```

## 実行フェーズ

| フェーズ | 期間 | アクション |
|---------|------|-----------|
| Phase 1 | 今すぐ | ずんだもんショートを月4本追加 → 登録者獲得 |
| Phase 2 | 1-2ヶ月後 | ガブリエルのキャラデザ＋VRM化 → テスト動画2本 |
| Phase 3 | 3ヶ月後〜 | アルカナチャンネル本格稼働 → 4天使掛け合い動画 |
| Phase 4 | 適時 | B2B向け動画に顔出しパターンを限定導入 |

---

## KPI

- **Phase 1**: ずんだもん動画からのチャンネル登録者数の増加率
- **Phase 2**: ガブリエル動画のCTR・視聴維持率（ずんだもん比較）
- **Phase 3**: アルカナキャラ動画のエンゲージメント率
- **全体**: ファネル下部への転換率（YouTube → STORES / B2B問い合わせ）
