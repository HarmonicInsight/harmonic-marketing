## ステータス定義

| ステータス | 意味 |
|-----------|------|
| `idea` | アイデア段階。タイトルやテーマのみ |
| `draft` | 下書き作成中 |
| `review` | レビュー待ち |
| `ready` | 公開準備完了 |
| `published` | 公開済み |
| `archived` | アーカイブ（非公開化・差し替え済み） |
