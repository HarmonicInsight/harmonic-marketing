# ニュース速報分析シリーズ テンプレート

## シリーズ概要

ニュースやリリースの「速報 → 独自分析・アンチテーゼ」を提供するコンテンツフォーマット。
タイムリー性を武器に、ニュースの表面だけでなく「本質は何か」を語る。

**チャネル**: YouTube Shorts / YouTube通常動画 / note / X
**更新頻度**: ニュースが出たら即対応（72時間以内が理想）

---

## サムネイル仕様（共通フォーマット）

### レイアウト

```
┌──────────────────────────────────┐
│  ┌────────────┐                  │
│  │  3月18日   │  ← 日付ラベル    │
│  │  リリース   │   （左上 or 右上）│
│  └────────────┘                  │
│                                  │
│   メインキャッチコピー            │
│   （2行以内、大きく太字）         │
│                                  │
│  ┌──────────┐                    │
│  │ ソースロゴ │ ← 発信元ロゴ/名  │
│  └──────────┘                    │
│                                  │
│           [背景画像/グラデ]       │
└──────────────────────────────────┘
```

### 日付ラベルの仕様
- **形状**: 角丸ラベル or リボン風
- **背景色**: 赤系（#E53935）or オレンジ系（#FF6D00）― 速報感を出す
- **文字色**: 白（#FFFFFF）
- **フォント**: ゴシック太字、16〜20pt相当
- **表記例**:
  - `3月18日 リリース`
  - `3月13日 発表`
  - `3月20日 公開`
  - `速報 3月18日`

### メインキャッチコピー
- **フォント**: ゴシック太字、28〜36pt相当
- **文字色**: 白 or 黒（背景に応じて）
- **行数**: 最大2行
- **例**:
  - `81,000人が答えた`
  - `AIに求めるもの`

### ソース表示
- 発信元の名前やロゴを小さく表示
- **例**: `Anthropic` `楽天` `経産省` `PR TIMES`

### 背景
- **Shorts**: ダーク系グラデーション（黒→濃紺）
- **通常動画**: ニュースソースのスクリーンショット（ぼかし加工）+ オーバーレイ
- **アクセントカラー**: ニュースの種類で変える
  - AI・テック系 → 青〜紫（#1565C0〜#6A1B9A）
  - DX・経営系 → 緑〜ティール（#00695C）
  - 政策・制度系 → 赤〜オレンジ（#C62828〜#E65100）

## Shorts台本テンプレート（45〜60秒）

```markdown
# [シリーズ名]：[キャッチコピー]

## メタ情報

- **プラットフォーム**: YouTube Shorts
- **尺**: 45〜60秒
- **シリーズ**: ニュース速報分析
- **サムネテキスト**: [2行以内のキャッチ]
- **日付ラベル**: [○月○日 リリース]
- **ソース**: [発信元名]
- **元URL**: [ニュースURL]
- **鮮度期限**: [公開から○日以内推奨]

## タイトル案

- [タイトル案1]
- [タイトル案2]

## 台本

### 冒頭フック（3秒）

「[ニュースの核心を疑問形 or 意外性で提示]」

### ニュース紹介（10秒）

「[何月何日、誰が、何を発表したか。事実を端的に]」

### 独自分析 / アンチテーゼ（30秒）

「[表面的な読み方ではなく、本質的に何を意味するか]
[データや具体例を使って深掘り]
[一般的な解釈と自分の見解の対比]」

### 締め（5秒）

「[1文で結論。記憶に残るフレーズ]」

## テロップ案

- [日付ラベル] [ソース名]
- [ニュースの要点]
- [核心データ]
- [アンチテーゼの結論]

## CTA

「[次の動画への誘導 or noteへの誘導]」

## 出典

- [ソース名] ([日付])
- [URL]
```

## 通常動画台本テンプレート（5〜12分）

```markdown
# [タイトル]

## メタ情報

- **プラットフォーム**: YouTube（通常動画）
- **尺**: [5〜12]分
- **シリーズ**: ニュース速報分析
- **サムネテキスト**: [メインキャッチ]
- **日付ラベル**: [○月○日 リリース]
- **ソース**: [発信元名]
- **元URL**: [ニュースURL]

## 台本

### オープニング（〜1分）
「[ニュースの概要を紹介。なぜこれが重要か]」

### セクション1：ニュースの中身（〜2分）
「[事実とデータの紹介。ソースに忠実に]」

### セクション2：独自分析（〜3分）
「[表面的な読み方 vs 本質的な意味]
[他のデータや事例と組み合わせた分析]」

### セクション3：アンチテーゼ / 問題提起（〜2分）
「[一般的に見落とされている視点]
[このニュースが本当に示していること]」

### セクション4：実務への示唆（〜2分）
「[視聴者が明日からどう行動を変えるべきか]
[具体的な提言]」

### エンディング（〜30秒）
「[まとめ、CTA]」

## テロップ・図表案
[セクションごとに必要なビジュアル]

## 概要欄テキスト案
[概要、タイムスタンプ、関連リンク、ハッシュタグ]

## 出典
[ソース一覧]
```

## 運用ルール

### 鮮度管理
- **即出し（24時間以内）**: Shorts 1本 + X投稿
- **速報分析（72時間以内）**: Shorts 2〜3本シリーズ
- **深堀り（1週間以内）**: 通常動画 + note記事
- **1週間超え**: 速報感を出さず「振り返り分析」として出す

### ニュース選定基準
1. **DX・AI・業務改革に関連**するニュース
2. **データが含まれている**（数字で語れる）
3. **表面的な報道と本質にギャップがある**（アンチテーゼが成立する）
4. **日本企業の実務に示唆がある**

### ポジショニング
- 単なるニュース紹介者ではない
- **「ニュースの裏を読む人」「本質を突く人」**としてのブランディング
- 常に「で、あなたの仕事にどう関係あるの？」を語る
