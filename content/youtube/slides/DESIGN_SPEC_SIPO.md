# SIPO Presentation Design Spec

SIPO_Analysis_Introduction.pptx (JP/EN) のデザイン設定記録。
新しいプレゼンテーション作成時にこのスペックを再利用する。

## スライド基本設定

| 項目 | 値 |
|------|-----|
| サイズ | 13.33 x 7.50 inches (16:9 ワイド) |
| テーマ | Office (Calibri) |
| 背景色 | `#121224` (ダークネイビー) |
| ヘッダーバー | `#EF4444` (赤) h=0.07in, スライド上端 |

## カラーパレット

### SIPO要素カラー

| 要素 | メインカラー | 用途 |
|------|-------------|------|
| **S** Source | `#F59E0B` (amber) | Oval, ヘッダーバー |
| **I** Input | `#0096C7` (blue) | Oval, ヘッダーバー |
| **P** Process | `#8B5CF6` (purple) | Oval, ヘッダーバー |
| **O** Output | `#10B981` (green) | Oval, ヘッダーバー |

### UI カラー

| 用途 | カラー | Hex |
|------|--------|-----|
| カード背景 | ダーク | `#24243E` |
| テキスト (主) | 白 | `#FFFFFF` |
| テキスト (補足) | ライトグレー | `#D0D0E0` |
| テキスト (ミュート) | グレー | `#808090` |
| 矢印・接続線 | ライトブルー | `#48CAE4` |
| ヘッダーバー | 赤 | `#EF4444` |
| ✕ マーク背景 | — | カード背景と同色 |
| ◎ マーク背景 | — | ヘッダーバー付きカード |

## フォントサイズ（JP版）

テーマデフォルトのCalibrを使用。明示指定しない場合はテーマサイズが適用される。

| 用途 | サイズ |
|------|--------|
| スライドタイトル | 28pt (テーマ依存) |
| Oval内文字 (S/I/P/O) | 22pt bold |
| 接続テキスト | 18pt bold |
| ボトムメッセージ | 16pt |
| ラベル (SIPO₁等) | 14pt |
| ブランド名 | 10pt |

## フォントサイズ（EN版）

英語はテキストが長くなるため、JP版より小さめのサイズを明示指定する。

| 用途 | サイズ |
|------|--------|
| タイトルスライド大見出し | 36pt |
| スライドタイトル | 22-28pt |
| サブタイトル・説明文 | 14-20pt |
| カード内テキスト | 11-12pt |
| Oval内文字 (S/I/P/O) | 22pt bold |
| 接続テキスト | 18pt bold |
| 結論・インサイト文 | 9-13pt |
| ボトムメッセージ | 13-16pt |
| ラベル (SIPO₁等) | 14pt |
| ブランド名 | 10pt |

## レイアウトパターン

### 4列カード (Slide 4, 5, 7)

```
|  S card  |  I card  |  P card  |  O card  |
```

| 項目 | 値 |
|------|-----|
| カード幅 | 2.55-2.75in |
| カード高 | 2.80-4.20in |
| カード間ギャップ | Arrow(0.35in) + 余白 |
| Oval サイズ | 0.85-1.15in |
| カード開始Y | 1.80-2.20in |
| ヘッダーバー | カード上端に同色 h=0.07in |

### Before/After比較 (Slide 6)

```
| IPOだけの場合  |  SIPOの場合  |
| (左半分 5.50in) | (右半分 5.50in) |
```

| 項目 | 値 |
|------|-----|
| 左右幅 | 5.50in each |
| Oval (ステップ) | 0.55in |
| ステップ間 | ↓ で接続 |
| 結果テキスト | y=5.90in |

### チェーン図 (Slide 8)

```
SIPO₁:  S₁ → I₁ → P₁ → O₁
              ↓ O₁ becomes S₂ ↓
SIPO₂:  S₂ → I₂ → P₂ → O₂
```

| 項目 | 値 |
|------|-----|
| Oval サイズ | 0.85in |
| 文字サイズ | 22pt bold |
| Row 1 Y | 1.80in |
| Row 2 Y | 4.80in |
| 接続帯 | RoundedRect `#24243E` 中央 |
| 接続テキスト | 18pt bold `#48CAE4` |
| 矢印 (row間) | Down Arrow 0.30x0.45in |

### ✕/◎ 比較表 (Slide 9)

```
| ラベル (11.00in幅)                      |
```

| 項目 | 値 |

| カード幅 | 5.20in |
| カード高 | 0.90in |
| 行間隔 | ~1.70in |
| ✕/◎ 記号幅 | 0.50in |

### 3ステップ (Slide 10)

## ナレーション (TTS/字幕) ルール

| 項目 | 基準 |
|------|------|
| 1行の文字数 | 20〜45文字 (半角=0.5) |
| 1スライドの行数 | 5〜8行 (目安) |
| XML lang属性 | JP: `kumimoji="1" lang="ja-JP" altLang="en-US"` |
| XML lang属性 | EN: `lang="en-US" altLang="ja-JP"` |
| dirty属性 | `dirty="0"` 必須 |

## ブランド表記

- 正式: **HARMONIC insight** (全スライド左下 y=7.05in)
- フォント: 10pt, color `#808090`
- 位置: (0.60, 7.05) 4.00x0.35in, 左寄せ
