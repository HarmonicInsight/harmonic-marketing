# 売上110億円なのになぜ潰れた？──占部組の倒産をキャッシュフローで読み解く

## ソース情報
- **元記事（TBS NEWS DIG）**: https://newsdig.tbs.co.jp/articles/-/2574214?display=1
- **帝国データバンク倒産速報**: https://www.tdb.co.jp/report/bankruptcy/flash/5215/
- **東京商工リサーチ速報**: https://www.tsr-net.co.jp/news/tsr/detail/1202699_1521.html
- **公開日**: 2026年3月31日

## 倒産企業情報
- **会社名**: 株式会社占部組（福岡県春日市）
- **創業**: 1933年1月（業歴90年超）
- **資本金**: 2,000万円
- **事業内容**: コンビニ等小売店舗の建築工事、造成工事
- **2025年8月期 完工高**: 110億8,710万円（過去最高）
- **負債総額**: 約49億9,414万円
- **事業停止日**: 2026年3月31日

---

## 「売上110億、負債50億」だけでは何もわからない

帝国データバンクや各メディアの報道では、占部組の倒産原因として「外注依存度が高い」「粗利益率が低い」「コンビニ出店減」「支払いトラブル」が挙げられている。しかし、これだけ読んでも**「なぜ売上110億円の会社が50億円の負債で潰れるのか」**は正直よくわからない。

倒産の本質は常に**キャッシュフローの破綻**である。売上がいくら大きくても、手元の現金が支払いに追いつかなければ会社は止まる。

占部組の詳細な財務諸表（B/S・P/L・CF計算書）は非上場企業のため公開されていない。そこで本記事では、**建設業の業界平均データ**を使い、売上110億円・負債50億円の建設会社で何が起きていたかをキャッシュフローの視点からシミュレーションする。

---

## シミュレーション：占部組のキャッシュフロー構造を推定する

### 前提条件

| 項目 | 業界平均 | 占部組の推定値 | 根拠 |
|---|---|---|---|
| 完工高（売上） | ─ | **110億円** | 帝国データバンク発表 |
| 外注費比率 | 31〜33%（※1） | **60〜70%**（推定） | 「外注依存度が高い」との報道。自社職人を持たず全国展開していた事業モデルから、業界平均の2倍程度と推定 |
| 資材費比率 | 約16% | **10〜15%**（推定） | 店舗建築主体のため資材比率はやや低めと想定 |
| 粗利益率 | 20〜25% | **15〜20%**（推定） | 「粗利益率が低水準」との報道 |
| 経常利益率 | 3.5〜4% | **1〜2%**（推定） | 「売上高経常利益率が低下」との報道 |
| 入金サイト | 60〜90日 | **90日**（推定） | 建設業の標準的な回収期間 |
| 支払サイト | 30〜60日 | **30〜45日**（推定） | 外注先への支払いは入金より先行 |

> ※1 日本政策金融公庫「業種別経営指標」建設業の外注費対完成工事高比率

---

### Step 1：月次の収支構造を推定する

**年間完工高110億円 → 月商 約9.2億円**

| 項目 | 月額（推定） | 対売上比率 |
|---|---|---|
| 完工高（売上） | 9.2億円 | 100% |
| 外注費 | ▲5.5〜6.4億円 | 60〜70% |
| 資材費 | ▲0.9〜1.4億円 | 10〜15% |
| 人件費・販管費 | ▲1.2〜1.5億円 | 13〜16% |
| **粗利益** | **1.4〜1.8億円** | **15〜20%** |
| **経常利益** | **0.1〜0.2億円** | **1〜2%** |

ここが最初のポイントだ。**月商9.2億円に対して、手元に残る経常利益は月1,000〜2,000万円程度**。年間でも1〜2億円しか利益が出ていない。

---

### Step 2：運転資金のギャップを計算する

建設業の資金繰りが厳しい最大の理由は、**「先にカネが出て、後からカネが入る」**構造にある。

```
【占部組のキャッシュフロー・タイムライン（推定）】

月0   月1   月2   月3   月4
 ├─────┼─────┼─────┼─────┤
 着工   外注費  外注費  竣工   入金
       支払①  支払②  検収   ←ここでやっと回収
 
 ← 先に出ていく：5.5〜6.4億円/月 →
                         ← 入ってくる →
```

#### 必要運転資金の試算

```
入金サイト90日（3か月分の売上が未回収）
  = 9.2億円 × 3か月 = 27.6億円

支払サイト45日（1.5か月分の外注費が未払い）
  = 6億円 × 1.5か月 = 9億円

必要運転資金 = 未回収売上 − 未払外注費
             = 27.6億円 − 9億円
             = 約18〜19億円
```

つまり、占部組は事業を回すだけで**常時18〜19億円の運転資金**を手元に持っているか、借入で調達している必要があった。

---

### Step 3：自己資本と借入の関係を見る

| 項目 | 金額 |
|---|---|
| 資本金 | 2,000万円 |
| 負債総額 | 約50億円 |
| 必要運転資金（推定） | 18〜19億円 |
| 年間経常利益（推定） | 1〜2億円 |

**資本金2,000万円で50億円の負債を抱えている**。この数字が意味するのは：

- 事業に必要な資金のほぼすべてが**借入金と買掛金（外注先への未払い）**で賄われていた
- 年間利益1〜2億円に対して、負債50億円は**返済に25〜50年かかる計算**
- 自己資本比率は極めて低く、**1件の大型トラブルでも即座に資金ショートする**構造

---

### Step 4：なぜ破綻したか──3つの圧力の同時発生

占部組の資金繰りを破綻に追い込んだのは、以下の3つの圧力が同時に発生したことだと推定される。

#### 圧力①：コンビニ出店減による売上の質の悪化

```
【コンビニ出店減の影響】

以前：九州エリアの店舗工事30%を担う安定受注
  → リピート案件で入金サイクルが安定
  → 発注元の信用力が高く、回収リスクが低い

現在：新規出店数が減少
  → 安定した「太い案件」が減る
  → 新分野（商業ビル・物流倉庫・福祉施設）は単発案件が多く、
     入金サイクルが読みにくい
  → 新規取引先は信用調査も不十分になりがち
```

売上の「総額」は110億円で過去最高だったが、**売上の「中身」が変わっていた**可能性が高い。安定したコンビニ工事から、不慣れな新分野への移行で、キャッシュフローの予測精度が落ちた。

#### 圧力②：外注費高騰による利益の蒸発

建設業界全体で人手不足が深刻化し、外注費は年々上昇している。

```
【外注費高騰のインパクト試算】

外注比率が65%→70%に5%上がると：
  110億円 × 5% = 5.5億円の利益減少

年間経常利益が1〜2億円しかない会社にとって
  → 5.5億円の原価上昇は致命的
  → 利益が完全に吹き飛ぶどころか赤字に転落
```

#### 圧力③：支払いトラブルによる信用毀損の連鎖

2023年頃から発生した取引先との支払いトラブルは、単なる「揉め事」ではなく、**資金繰り破綻の引き金**となった。

```
【破綻の連鎖メカニズム（推定）】

支払いトラブル発生（数億円規模）
  ↓
取引先が支払いを拒否 or 訴訟（2025年12月）
  ↓
入金予定だったキャッシュが止まる
  ↓
外注先への支払いが遅延
  ↓
外注先が施工を拒否 → 工期遅延 → さらに入金が遅れる
  ↓
銀行が追加融資に慎重になる
  ↓
資金ショート → 事業停止（2026年3月31日）
```

運転資金に余裕がない（自己資本がほぼゼロに等しい）状態では、**1件の支払いトラブル（数億円規模）で連鎖的に全体が崩壊する**。これが売上110億円の会社が倒産した本当のメカニズムである。

## まとめ：この倒産から何を読み取るべきか

占部組の倒産を一言でまとめると：

> **月商9億円・月の利益1,000万円・常時18億円の運転資金が必要──この薄氷の上を、資本金2,000万円で渡っていた**

「売上110億、負債50億」という数字だけでは見えない本質は、ここにある。

### 自社の資金繰りをチェックする3つの問い

1. **月次キャッシュフローは黒字か？**
   - 売上ではなく、「毎月、手元の現金が増えているか減っているか」を見る
   - 外注費・資材費の支払いタイミングと入金タイミングのギャップは何日か

2. **運転資金の何か月分の現預金を持っているか？**
   - 建設業なら最低3〜6か月分の運転資金を現預金で確保すべき
   - 占部組の場合、18億円 × 3か月 = 54億円が理想だが、実態は負債50億円

3. **1件の大型トラブルに耐えられるか？**
   - 月間利益1,000万円の会社が、数億円の支払いトラブルに耐えられるわけがない
   - 「最悪のシナリオ」を常に想定し、その時に何か月持つかを計算しておく

## 補足：本記事の限界について

本記事のキャッシュフロー分析は、**建設業の業界平均データと公開報道に基づく推定値**である。占部組は非上場企業であり、詳細な財務諸表（損益計算書、貸借対照表、キャッシュフロー計算書）は公開されていない。

実際の外注比率、利益率、借入金の内訳、入金サイトなどは推定値と異なる可能性がある。ただし、「売上110億円・負債50億円・資本金2,000万円・外注依存度が高い・経常利益率が低下」という確定情報から導かれるキャッシュフロー構造の脆弱性は、推定値の多少のズレでは変わらない。
