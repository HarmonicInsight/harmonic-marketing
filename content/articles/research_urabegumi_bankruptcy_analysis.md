# 「売上110億円の建設会社が倒産」──ニュース記事を読み解くと、資金繰りの本当の怖さが見えてくる

## ソース情報

- **帝国データバンク倒産速報**: https://www.tdb.co.jp/report/bankruptcy/flash/5215/
- **東京商工リサーチ速報**: https://www.tsr-net.co.jp/news/tsr/detail/1202699_1521.html

---

## はじめに

2026年3月31日、福岡県の総合建設会社「占部組」が事業を停止し、自己破産申請の準備に入りました。売上高110億円、創業90年超の老舗企業です。

このニュース、帝国データバンクや各メディアの報道を読むと「外注依存度が高い」「コンビニ出店減」「支払いトラブル」といった言葉が並びます。でも正直、**これだけ読んで「なるほど、だから潰れたのか」と腹落ちする人は少ない**のではないでしょうか。

本記事では、報道で使われている一つひとつの表現を取り上げ、**「これ、キャッシュフロー的にはこういう意味です」**と読み解いていきます。

---

## 報道の原文を一文ずつ読み解く

### 原文①：「負債総額は約50億円」「売上高110億円の総合建設会社が事業停止」

まず多くの人が「売上110億もあるのに、50億の借金で潰れるの？」と思うはずです。

でも大事なのは**売上ではなく、毎月いくら手元に現金が残るか**です。

```
年間売上110億円 → 月商 約9.2億円
```

ここから後で見るように、手元に残る利益はごくわずか。売上の大きさと「支払い能力」はまったくの別物です。

---

### 原文②：「外注依存度が高く粗利益率が低水準で推移」

これは占部組の**ビジネスモデルそのもの**の説明です。

占部組はコンビニの店舗建築を全国展開するために、各地の協力業者に施工を外注していました。自社で職人を何百人も抱えるのではなく、外注で回すモデルです。

これ自体は経営判断であって、間違いとは言えません。ただし結果として：

```
建設業の外注比率の平均：31〜33%（日本政策金融公庫 業種別経営指標）
占部組の外注比率（推定）：60〜70%（「依存度が高い」との報道から推定）

月商9.2億円のうち、5.5〜6.4億円が外注費として出ていく
→ 手元に残る粗利は月1.4〜1.8億円程度（粗利率 約15〜20%）
```

**月商9億円に見えて、実質は月1.5億円程度の粗利で会社を回している**。ここをまず押さえてください。

---

### 原文③：「近年はコンビニの新規出店数の減少を受けて売上高経常利益率が低下」

ここ、実はこの記事で一番重要な一文です。よく読んでください。

**「営業利益率」ではなく「経常利益率」が低下**と書いてあります。

この違いが何を意味するか：

```
売上
 − 原価（外注費・資材費）
 − 販管費（人件費・家賃等）
＝ 営業利益  ← 「本業の儲け」

 − 支払利息（借入金の利息）  ← ★ここで削られている
＝ 経常利益
```

もし本業の収益力が落ちたなら「営業利益率が低下」と書くはずです。わざわざ「経常利益率」と書いてあるということは、**借入金の利息負担が重くなっていた**ことを示唆しています。

#### なぜ借入が膨らんだのか

建設業は「先にカネが出て、後からカネが入る」構造です。

```
外注費の支払い：工事中〜完了後30〜45日
売上の入金：竣工・検収後60〜90日

→ 常に1.5〜3か月分のギャップが発生
→ 月商9.2億円 × 3か月 = 約27億円が常に「未回収」
→ この間の支払いを借入で賄う必要がある
```

売上が伸びれば伸びるほど、この立替資金も膨らみます。2024年8月期に100億円を突破し、2025年8月期に110億円と急成長した裏で、**運転資金のための借入も急増していた**はずです。

仮に負債50億円に対して金利2〜3%とすると：

```
年間の支払利息 = 50億円 × 2〜3% = 1〜1.5億円
```

粗利が月1.5億円、年間18億円。そこから販管費を引いた営業利益が仮に年2〜3億円だとすると、**利息だけで利益の3〜5割が飛ぶ**。

コンビニ出店が減ったこと自体よりも、**その穴を埋めるために新分野に進出→不慣れな案件で運転資金がさらに必要→借入増加→利息増加→経常利益率が低下**という連鎖が起きていたと読み取れます。

---

### 原文④：「商業ビルや物流倉庫、福祉施設といった新分野の開拓や関東・関西エリアへの営業強化」

一見すると前向きな多角化に見えますが、キャッシュフローの視点で読むと**危険信号**です。

```
コンビニ店舗工事（従来の主力）：
  ・発注元は大手チェーン → 信用力が高く入金が確実
  ・リピート案件 → 見積精度が高く、追加コストが出にくい
  ・パターン化された施工 → 外注管理もしやすい

新分野（商業ビル・物流倉庫・福祉施設）：
  ・新規取引先 → 信用調査が不十分になりがち
  ・初めてのタイプの工事 → 見積精度が低く、コスト超過リスク
  ・案件単価が大きい → 1件のトラブルが致命傷になる
```

さらに「関東・関西エリアへの営業強化」は、**営業所の開設・維持コスト（固定費）の増加**を意味します。売上が不安定な新エリアに固定費を投じるのは、資金繰りが厳しい状況ではリスクが大きい。

---

### 原文⑤：「2023年頃から取引先との認識相違に基づく支払いトラブルが一部現場で発生」

「認識相違に基づく支払いトラブル」──これは建設業でよくある表現ですが、要するに**「工事の範囲や仕様で揉めて、代金を払ってもらえなかった」**ということです。

この一文がなぜ致命的だったか：

```
月の経常利益：推定1,000〜2,000万円
年間の経常利益：推定1〜2億円

支払いトラブル（数億円規模と推定）
  → 年間利益の数倍のキャッシュが止まる
  → 外注先への支払いは待ってくれない
  → 2025年12月に訴訟提起 → 解決に時間がかかる
  → その間もキャッシュは出ていく一方

資本金2,000万円 → バッファがほぼゼロ
  → 数億円のトラブル1件で詰む
```

年間利益1〜2億円の会社にとって、数億円規模の未回収は**即死レベル**です。大企業なら「今期は特損を計上して…」で済む話が、自己資本がほぼない会社では資金ショートに直結します。

## 全体像：占部組に何が起きていたか

報道の断片を組み合わせると、こういうストーリーが浮かび上がります。

```
【第1段階】薄利多売モデルの確立
  外注主体で全国展開 → 粗利は薄いが売上は大きい
  コンビニの安定受注で回転 → 資金繰りも予測可能

【第2段階】前提の崩壊
  コンビニ新規出店が減少 → 太い安定案件が減る
  売上維持のため新分野に進出 → 固定費増加・リスク増大

【第3段階】借入依存の深化
  新分野の運転資金が必要 → 借入増加
  借入の利息が営業利益を圧迫 → 経常利益率が低下
  ← 報道の「経常利益率の低下」はここを指している

【第4段階】トリガーの発生
  取引先との支払いトラブル → 数億円のキャッシュが止まる
  年間利益1〜2億円の会社にとって致命傷
  2025年12月の訴訟で回収の見通し立たず

【第5段階】連鎖的な崩壊
  借入返済 + 利息 + 外注費の支払い > 入金
  銀行の追加融資も困難に
  2026年3月31日 事業停止
```

## この事例から考える「自社の資金繰り」チェックリスト

占部組の事例を他人事にしないために、自社でチェックすべきポイントを3つに絞ります。

### ✅ 1. 月次キャッシュフローを把握しているか

売上と利益だけでなく、**毎月の入金額と出金額、その差額**を見る。建設業なら入金サイトと支払サイトのギャップ日数を正確に把握しているかが生命線。

### ✅ 2. 借入の利息が利益を食っていないか

損益計算書の「営業利益」と「経常利益」の差額を見る。この差が大きくなっていたら、**借入コストが利益を侵食している**サイン。占部組はまさにここが悪化していた。

### ✅ 3. 最悪の1件に耐えられるか

「最大の取引先が3か月支払いを止めたら？」「最大の工事で数億円の追加コストが出たら？」──その時、手元資金で何か月持つかを計算する。答えが「1か月以下」なら、占部組と同じ構造的リスクを抱えている。
