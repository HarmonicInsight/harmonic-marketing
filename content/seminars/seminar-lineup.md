## セミナーA: SIPOフレームワーク実践ワークショップ

### 基本情報

| 項目 | 内容 |
|------|------|
| **ターゲット** | 建設業・製造業の経営企画、DX推進、業務改革担当 |
| **価格** | ¥15,000（半日 3時間） |
| **定員** | 15名 |
| **開催形式** | オンライン（Zoom）|
| **頻度** | 月1回 |
| **集客元** | VID-025（SIPO解説）、NOTE-P001（SIPO実践ガイド）、note無料記事 |

### カリキュラム（3時間）

| 時間 | 内容 | 形式 |
|------|------|------|
| 0:00-0:30 | IPOの限界とSIPOの必要性 | 講義 |
| 0:30-1:30 | 自社の業務のSourceを特定する | ワーク（個人→グループ共有） |
| 1:30-1:40 | 休憩 | - |
| 1:40-2:40 | 4層モデルで自社バックオフィスを再設計する | ワーク（テンプレート使用） |
| 2:40-3:00 | 質疑応答 + 個別相談案内 | Q&A |

### 参加者が持ち帰れるもの
- 自社業務のSource一覧（ワーク成果物）
- 4層モデルの初期設計（ワーク成果物）
- SIPOテンプレート（Excel/PDF）
- 個別相談30分の無料枠（コンサル導線）

### コンサル転換の仕掛け
- ワーク中に「ここは自社だけでは判断が難しい」ポイントが必ず出る
- そこで「個別相談（無料30分）でお手伝いできます」と自然に案内
- 個別相談 → コンサル案件化

## セミナーB: AI×プロトタイプ駆動型コンサルティング実践講座

### 基本情報

| 項目 | 内容 |
|------|------|
| **ターゲット** | 企画職・PM・コンサルタント（40〜50代中心） |
| **価格** | ¥15,000（半日 3時間） |
| **定員** | 12名（ハンズオンのため少人数） |
| **開催形式** | オンライン + ハンズオン |
| **頻度** | 月1回 |
| **集客元** | VID-022（プロトタイプ駆動提案）、NOTE-P002（Claude Code実践ガイド） |

### 事前準備（参加者）
- Claude Codeのアカウント作成（無料プランでOK）
- 自社の業務課題を1つ選んでくる（「これをプロトタイプにしたい」テーマ）

### カリキュラム（3時間）

| 時間 | 内容 | 形式 |
|------|------|------|
| 0:00-0:20 | プロトタイプ駆動型コンサルティングとは | 講義 |
| 0:20-0:50 | Claude Code基礎 — セットアップと基本操作 | ハンズオン |
| 0:50-1:00 | 休憩 | - |
| 1:00-1:20 | 要件定義をプロンプトに変換する方法 | 講義+演習 |
| 1:20-2:20 | 実践ワーク: 自社課題をプロトタイプ化する | ハンズオン（個別サポート） |
| 2:20-2:50 | 成果物レビュー（全体で共有・フィードバック） | グループ |
| 2:50-3:00 | まとめ + 次のステップ案内 | - |

### 参加者が持ち帰れるもの
- 自社課題のプロトタイプ（実際に動くもの）
- 要件定義→プロンプト変換テンプレート
- Claude Code設定チートシート
- STORESツールの20%割引クーポン

### 製品転換の仕掛け
- ワーク中にSTORESのツールを「こういう時にこのツールが便利です」と自然に紹介
- 割引クーポンで購入のハードルを下げる
- 「もっと本格的にやりたい」→ コンサル問い合わせ

| 時間 | 内容 | 形式 |

## セミナー共通: 運営設計

### 集客チャネル

| チャネル | 施策 | 期待効果 |
|---|---|---|
| YouTube本編 | 動画末尾の3段階CTA + 概要欄リンク | メイン流入 |
| YouTube Shorts | 課題提起 → 本編 → セミナー（2段階） | 認知→興味 |
| note無料記事 | 記事末尾にセミナー告知 | SEO経由の長期流入 |
| note有料記事 | 記事内で「もっと深くやるならセミナー」 | 有料読者=高転換率 |
| Twitter | セミナー告知 + 参加者の声 | リマインド |
| メール | セミナー参加者への次回案内 | リピーター |

### 価格設計の考え方

| セミナー | 価格 | 根拠 |
|---|---|---|
| A（SIPO） | ¥15,000 | 半日ワークショップ。成果物持ち帰り。コンサル導線価値 |
| B（プロトタイプ） | ¥15,000 | ハンズオン。少人数。プロトタイプ持ち帰り |
| C（INMV） | ¥10,000 | 2時間。製品トライアル付き。製品転換が主目的なので低価格 |
| 法人研修 | ¥100,000-¥300,000 | 企業別カスタマイズ。セミナー参加後のアップセル |

### セミナー → コンサル転換フロー

```
セミナー参加（¥10,000-¥15,000）
  ↓
個別相談（無料30分 — セミナー特典）
  ↓
課題の深堀り（ここでコンサルニーズが顕在化）
  ↓
コンサル提案（案件ベース）
  ↓
コンサル契約
```

| 項目 | コスト | 備考 |
